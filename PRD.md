## 一、项目概述

### 1.1 项目名称
**Yahao Digest** — Yahao 的个人文摘网站

### 1.2 一句话定位
一个以**经典英文报纸 Digest（文摘）**为视觉基底的个人网站，黑白极简、衬线字体主导、手绘插画配图，用于展示文章、摄影作品和生活感悟。

### 1.3 网站主人画像
- **姓名/笔名**: Yahao
- **背景**: 新闻传播学出身，现为美妆个护产品经理，工作 15 年+
- **个人介绍**: "前新闻人，后产品经理。工作15年+，用文字和摄影记录人生。"
- **技术水平**: 非技术背景，需要内容管理方便直觉化
- **摄影风格**: ins 风格为主，辅以产品渲染图

### 1.4 目标受众
- 对美妆个护行业感兴趣的读者
- 喜欢阅读深度内容和生活感悟的文艺群体
- 摄影爱好者
- 潜在合作品牌/雇主

## 三、设计风格定义

### 3.1 色彩系统

| 色彩角色 | 色值 | 用途 |
|---------|------|------|
| **纸张背景** | `#FAF7F0` ~ `#F5F0E8` | 全站背景，模拟旧报纸纸张 |
| **内容区背景** | `#FFFFFF` / `#FEFCF8` | 文章卡片、内容区域 |
| **主文字** | `#1A1A1A` | 标题、正文 |
| **次要文字** | `#6B6B6B` | 署名、日期、摘要 |
| **装饰线** | `#2A2A2A` | 分隔线、边框 |
| **强调色** | `#8B0000`（暗红） | 极少量使用：悬停状态、分类标签 |
| **链接悬停** | `#333333` | 文字悬停加深 |

#### 🌙 暗夜模式（胡桃木色系）

支持浅色/深色模式切换。暗夜模式采用**胡桃木深褐色**基调，而非传统的灰黑暗夜：

| 色彩角色 | 色值 | 用途 |
|---------|------|------|
| **背景色** | `#2C1E14` ~ `#3A2A1E` | 胡桃木深褐色主背景 |
| **内容区背景** | `#3E2D20` / `#4A372A` | 卡片/内容区域，稍浅的胡桃色 |
| **主文字** | `#F0E6D8` | 标题、正文（暖白/米色） |
| **次要文字** | `#B8A898` | 署名、日期、摘要 |
| **装饰线** | `#8B7355` | 分隔线、边框（古铜色调） |
| **强调色** | `#C4836A` | 悬停状态（暖陶土色） |
| **纸张纹理** | 暗色木纹/皮革纹理 | 微妙的胡桃木质感 |

模式切换按钮位于导航栏右侧，样式为复古油灯/蜡烛图标。

### 3.2 字体系统

| 层级 | 字体 | 大小 | 粗细 | 用途 |
|------|------|------|------|------|
| **报头** | Playfair Display | 48-64px | Bold (700) | "Yahao Digest" |
| **署名手写** | 手写体（Dancing Script / Caveat） | 24-28px | Normal | 右上角 "Yahao" |
| **引言** | Playfair Display / Noto Serif SC | 24-32px | Normal Italic | 每期精选语录 |
| **文章标题** | Noto Serif SC / Playfair Display | 18-22px | Bold (700) | 文章标题 |
| **署名** | Noto Sans SC | 14px | Normal | "By Yahao" 等 |
| **正文/摘要** | Noto Serif SC | 14-16px | Normal (400) | 文章正文、摘要 |
| **元信息** | Noto Sans SC | 12-13px | Normal | 日期、阅读时间等 |

### 3.3 头像设计
- **样式**: 人物剪影（silhouette）
- **用途**: 关于我页面、侧边栏作者简介、个人名片弹窗
- **暗夜模式**: 剪影颜色自适应（浅色模式黑色剪影，暗夜模式暖白剪影）

### 3.4 光标设计
- **默认光标**: 自定义鹅毛笔 SVG 图标（32x32），黑色笔身
- **链接悬停**: 鹅毛笔倾斜/蘸墨状态
- **点击**: 鹅毛笔按下状态
- **暗夜模式**: 鹅毛笔颜色切换为暖白/米色

### 3.5 纸张纹理
- 全站 `body` 应用淡米黄纸张纹理（CSS 噪点 + 渐变）
- 保持参考图的干净感，纹理要非常克制、若隐若现
- 内容卡片区域用纯白/微暖白底
- **暗夜模式**: 纹理切换为深色木纹/皮革质感

### 3.6 网站语言
- **主要语言**: 中文
- UI 文案、导航、按钮、日期格式等均使用中文
- 报头 "Yahao Digest" 保持英文（品牌名）
- 日期格式：中文模式 "2026年2月19日 星期四"（浅色/暗夜均适用）

## 五、功能模块详细设计

### 5.1 报头区（Masthead）
**完全参考 Twitter Digest 排版：**
- 左侧："Yahao Digest"，Playfair Display Bold，超大号
- 右侧上方："Yahao"，手写体（Dancing Script），作为署名
- 右侧下方：当前日期 "February 19, 2026" 格式
- 报头下方：1px 黑色装饰细线，全宽

### 5.2 导航栏（Navigation）
- 紧贴报头下方，水平排列
- **栏目**: 首页 · 深度报道 · 美妆洞察 · 生活随笔 · 摄影集 · 关于
- 字体：小号衬线体，字间距宽松（letter-spacing: 2px）
- 右侧：搜索图标（放大镜），点击展开搜索输入框
- 分隔符：圆点（·）分隔各导航项
- 导航栏下方再加一条细线

### 5.3 引言区（Featured Quote）
**参考图核心特征之一：**
- 居中排版，上下留白充足
- 大号斜体衬线字体
- 内容：每期精选一句话（名人名言/书摘/自己的金句）
- 引号使用大号花引号 " "
- 来源格式："— 作者名 on 来源"
- 下方装饰性分割线（可以是双线或花纹线）

### 5.4 头条新闻滚动条（Breaking News Ticker）
- 位于引言区下方
- 可左右滑动，展示 3~5 篇最新专题
- 每个头条包含：标题 + 分类标签 + ⏳ 阅读时间
- 视觉：与整体黑白风格统一，不使用彩色背景
- 两侧箭头导航，自动轮播 5 秒

### 5.5 文章网格区（Article Grid）
**核心参考 Twitter Digest 三栏布局：**
- **三栏等宽**网格，卡片之间用**细竖线**分隔
- 每个文章卡片包含：
  1. **配图**：方形/横向比例，使用示例 stock 图片（Unsplash/Pixabay）
  2. **标题**：衬线字体加粗，悬停触发**墨水晕染效果**
  3. **署名**："By Yahao"，小号灰色
  4. **摘要**：2-3 行，小号正文，灰色
  5. **⏳ 阅读时间**（可选）
- **文章仅包含标题和摘要**，不展示完整正文内容（MVP 阶段）
- **内容比例**: 文章 80% + 摄影 20%
  - 每展示 6 篇文章（2 行）后，插入 1 组摄影作品快照

### 5.6 摄影穿插区（Photo Insert）
- 在文章网格之间穿插
- 横向 2~3 张照片小型网格
- 每张带简短标注
- 底部链接："查看完整摄影集 →"

### 5.7 侧边信息区
- 位于首页底部（非传统侧边栏，保持三栏式主布局的纯净感）
- 或在宽屏下作为右侧窄栏出现
- 包含：
  - 📅 当日日期（复古日历小组件）
  - 🌤 天气（装饰性固定数据）
  - 🏷 热门标签云
  - 👤 作者简介缩略

### 5.8 文章详情页
> **MVP 阶段说明**: 文章详情页仅展示标题 + 摘要，不展示完整正文。后续有正式内容后再扩展全文排版。

- **报纸风格排版**:
  - 大标题居中，Noto Serif SC Bold
  - 署名："By Yahao · 2026年2月19日"
  - 摘要段落，衬线字体
  - 配图展示
- **折痕分割线**: 标题与摘要之间的装饰性分割
- **⏳ 阅读时间**: 顶部沙漏图标 + 分钟数
- **评论区**: 手写体风格（详见 5.11）
- **文章分类标签**: 顶部显示
- **后续扩展**: 正文两端对齐、首字母下沉（Drop Cap）、长文段落折痕分割

### 5.9 摄影集页面
- 网格式布局展示所有摄影作品
- 每张照片包含：
  - 仿旧胶片编号 "FILM No. 001"
  - 手绘风格图说标注
  - 轻微做旧滤镜
- **灯箱模式**（点击放大）:
  - 老照片不规则白边
  - 颗粒感噪点滤镜
  - 快门参数面板：`f/2.8 · 1/125s · ISO 400 · 35mm`
  - 拍摄背后故事文字
  - 触发快门声效

### 5.10 页脚（Footer）
- 模仿报纸版权声明
- 居中排版：
  - "© 2026 Yahao Digest. All Rights Reserved."
  - RSS 订阅 · 付费订阅（预留）
  - 复古印章图标
- **印章交互**: 点击 → 弹出个人名片
  - 名片风格：旧式名片纸质感
  - 内容：姓名、职位、介绍、微信号（预留）

### 5.11 评论区
- **手写体风格** UI
- 背景模拟旧报纸边角空白
- 输入框手写字体 + 墨迹装饰
- 标题："在报纸边角留下你的感悟"
- MVP 先做 UI + 示例评论，后续对接后端

### 5.12 声效系统
- **核心声效**: 摄影快门声（轻柔、清脆）
  - 触发：摄影灯箱打开时
- **声效控制**: 右下角声音开关图标，默认开启
- 文件体积 < 50KB，预加载

## 六、交互设计

### 6.1 墨水晕染效果（Ink Bleed Hover）
- **触发**: 鼠标悬停文章标题
- **表现**: 文字颜色微妙加深，周围出现柔和墨水扩散光晕
- **实现**: CSS `text-shadow` 过渡，300ms ease-out
- 保持黑白风格，不引入彩色

### 6.2 老照片效果（Vintage Photo）
- **触发**: 摄影作品点击放大
- **表现**: 不规则白色毛边 + 颗粒感噪点 + 轻微暖色调
- **退出**: 点击外部关闭

### 6.3 鹅毛笔光标
- 全站自定义光标，黑色鹅毛笔 SVG
- 正常：竖直 / 悬停可点击：倾斜

### 6.4 印章 → 名片弹窗
- 页脚印章图标点击
- 弹出浮层：旧式名片纸质感卡片
- 入场动画：印章盖下的动效感
